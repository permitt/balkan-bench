## Owner-action shortlist

1. Run the dataset publish (gate 1).
2. Run the 5-seed BERTić reproducibility gate on GCP (gate 5).
3. Run the L4 throughput sweep (gate 7).
4. Push the branch, merge to `main` once CI is green (gate 11).
5. Connect Vercel, verify the deploy, set the custom domain (gates 9-10).
6. Flip the repo to public and tag `v0.1.0` (gate 14).

## Deferred-to-v0.2

- Re-run all 9 models through the new Docker-based pipeline (current v0.1
  artifacts are seeded from the prior-pipeline aggregates).
- Croatian / Montenegrin / Bosnian language coverage.
- Serbian-LLM-Eval (Aleksa Gordić) adapter.
- MTEB-BCMS, LLM Arena.
- Multi-job GCP orchestrator; automated submission-scoring server.
- Prediction-level bootstrap CIs.
